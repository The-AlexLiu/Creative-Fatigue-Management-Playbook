# Creative Fatigue Management Playbook

**素材衰退管理方法：从爆款依赖到素材组合运营的经验沉淀**

过去看广告素材衰退时，很容易把问题理解成“这条素材跑了多少天”“频次是不是太高”“是不是该换一批新素材了”。

这些问题都重要，但如果只围绕单条素材讨论，素材管理很容易变成救火：

- 今天 ROAS 掉了，就怀疑素材疲劳；
- 明天 CPA 高了，就想暂停老素材；
- 后天新素材没跑起来，又把预算推回老素材；
- 最后账户看起来还在出单，但其实越来越依赖少数几个爆款。

我后来更倾向于先问另一个问题：

> 这个账户的 GMV，是由一组健康的素材组合支撑，还是被少数老素材硬撑着？

这个问题比“某条素材是不是老了”更重要。

因为真实投放里，老素材不一定坏。很多老素材依然能稳定赚钱。真正危险的是：头部素材开始衰退，但账户没有新素材接力；或者少数素材贡献了大部分 GMV，一旦掉量，整个账户就会跟着波动。

所以我现在看素材衰退，不会先问“第几天该换素材”，而是先看：

- 主力 GMV 是不是由老素材贡献；
- 预算是不是还压在已经衰退的素材上；
- 用户是不是反复看到同一批素材；
- Top1 / Top3 素材对 GMV 的贡献是否过度集中；
- 近 3 日表现相对近 7 日有没有真实下滑；
- 新素材补给是否能接住下一轮放量；
- 这次问题到底是素材疲劳、受众饱和，还是落地页和 offer 承接出了问题。

这套方法主要服务 DTC 独立站、Shopify 电商、跨境电商和以 GMV、ROAS、CPA 为核心目标的 Meta Ads、TikTok Ads、Google Demand Gen / Performance Max 投放。它不是一个单纯用来放代码的仓库，而是一个用来沉淀素材管理认知、判断框架和复盘模板的作品集项目。

---

## 核心判断

素材衰退管理的重点，不是追求每条素材永远不衰退，而是让账户始终有下一批素材可以接力。

我更认可的素材管理方式，是把素材当成一个组合来运营：

1. 老素材负责稳定现金流；
2. 新素材负责寻找下一批增长机会；
3. 变体素材负责延长赢家生命周期；
4. 数据指标负责判断到底是疲劳、饱和，还是漏斗问题；
5. 素材库负责把一次投放结果沉淀成下一轮创意资产。

落到日常看板里，我会把素材管理拆成 5 个模块：

| 模块 | 我主要看什么 | 它回答的问题 |
|---|---|---|
| 年龄结构 | GMV 加权日龄、花费加权日龄、曝光加权日龄 | 账户是不是被老素材撑住 |
| 曝光疲劳 | Frequency、Reach 增速、daily new users | 用户是不是反复看到同一批素材 |
| 表现衰减 | 近 3 日 vs 近 7 日 ROAS、CPA、CTR、CVR | 素材效率是不是真的在变差 |
| 集中风险 | Top1 GMV 占比、Top3 GMV 占比、HHI、有效素材数 | GMV 是否过度依赖少数爆款 |
| 补给能力 | 新增素材数、测试通过素材数、可替换素材库存天数 | 下一轮放量有没有素材接力 |

这 5 个模块要放在一起看。

如果只看日龄，容易误杀老赢家。  
如果只看 ROAS，容易忽略头部集中风险。  
如果只看 Frequency，容易把受众饱和误判成素材疲劳。  
如果只看新增素材数量，容易把同质化变体误认为素材供给充足。

## 我为什么要做这套方法

在真实投放里，素材衰退通常不是突然发生的。

它更像一个逐步累积的过程：

1. 账户先跑出 1-3 条主力素材；
2. 系统开始把更多预算分给这些赢家；
3. GMV 和 ROAS 短期看起来稳定；
4. 新素材因为历史数据弱，很难拿到足够曝光；
5. 主力素材日龄、花费占比、曝光占比持续上升；
6. CTR、CVR 或 ROAS 开始缓慢回落；
7. 团队发现问题时，已经没有足够成熟的新素材可以替换。

所以素材衰退管理真正要解决的，不是“哪条素材今天该关”，而是三个更底层的问题：

| 问题 | 如果不解决会怎样 |
|---|---|
| 素材组合是否过度集中 | 爆款一掉，GMV 和 ROAS 大幅波动 |
| 主力素材是否已经进入衰退趋势 | 预算继续加压，边际 CPA 越来越高 |
| 新素材补给是否足够 | 老素材掉量时没有接力素材 |

我之前用过两个指标来管理这个问题：

```text
GMV 加权日龄 = SUM(日龄 * GMV) / SUM(GMV)
Top3 GMV 占比 = Top3 素材 GMV 之和 / 总 GMV
```

如果你第一次看这些公式，可以先读 [指标公式解释](./docs/formula-explainer.md)。里面用具体数字解释了 GMV 加权日龄、花费加权日龄、曝光加权日龄、Top3 GMV 占比和 HHI。

这两个指标方向是对的。

但经过重新审视后，我认为它们更适合做“素材组合风险预警”，不能单独决定素材是否暂停。原因是：

- GMV 加权日龄会低估“花费高但 GMV 已经掉下去”的素材；
- Top3 GMV 占比只能说明集中度，不能说明头部素材是否健康；
- 老素材不一定疲劳，新素材也不一定健康；
- 素材表现下滑可能来自页面、价格、库存、归因或受众饱和。

因此，这套方法在原有两个指标上增加了花费加权日龄、曝光加权日龄、HHI、有效素材数、近 3/7 日衰减指标和动作分层。

## 我会先看 6 个问题

### 1. GMV 是不是被少数素材撑住

我不会只看总 ROAS。

如果账户整体 ROAS 还可以，但 Top1 或 Top3 素材贡献了大部分 GMV，账户其实已经有单点风险。

| 指标 | 我会怎么理解 |
|---|---|
| Top1 GMV 占比 > 30% | 单条素材对账户影响过大 |
| Top3 GMV 占比 > 60% | GMV 高度依赖头部素材 |
| Top3 GMV 占比 > 70% | 高风险，需要立刻补同角度变体 |
| 有效素材数 < 3 | 表面有很多素材，实际贡献集中在少数素材 |

这里的重点不是看到集中就马上拆掉赢家。

相反，赢家应该保留。真正应该做的是：围绕赢家背后的购买理由、视觉结构、达人类型、hook 和 offer，补一批可以接力的变体。

### 2. 老素材是在贡献 GMV，还是只是在吃预算

我会同时看三个加权日龄：

| 指标 | 公式 | 我主要用来判断什么 |
|---|---|---|
| GMV 加权日龄 | `SUM(日龄 * GMV) / SUM(GMV)` | 当前成交是否主要来自老素材 |
| 花费加权日龄 | `SUM(日龄 * Spend) / SUM(Spend)` | 当前预算是否主要压在老素材上 |
| 曝光加权日龄 | `SUM(日龄 * Impressions) / SUM(Impressions)` | 用户是否主要看到老素材 |

这三个指标放在一起才有意义。

| 表现 | 我会怎么判断 | 动作 |
|---|---|---|
| GMV 加权日龄高，ROAS 稳定 | 老赢家还健康 | 保留，补同角度变体 |
| 花费加权日龄高，GMV 加权日龄低 | 老素材吃预算但贡献下降 | 查预算浪费，考虑降预算 |
| 曝光加权日龄高，CTR 下滑 | 重复曝光压力上升 | 刷新 hook、封面、前三秒 |
| 三个日龄都上升，新素材贡献低 | 素材供给断层 | 优先补素材，不急着硬放量 |

### 3. 表现下滑是不是持续趋势

单日 ROAS 或 CPA 波动很正常。

我更愿意用近 3 日和近 7 日对比：

| 指标 | 公式 | 我会怎么用 |
|---|---|---|
| ROAS 衰减率 | `近3日ROAS / 近7日ROAS - 1` | 判断近期收入效率是否下滑 |
| CTR 衰减率 | `近3日CTR / 近7日CTR - 1` | 判断素材吸引力是否下降 |
| CPA 上升率 | `近3日CPA / 近7日CPA - 1` | 判断获客成本是否恶化 |

如果一条主力素材 GMV 占比高，但 ROAS 衰减超过 20%，CPA 同时明显上升，我不会继续把它简单归为“放量素材”。更合理的判断是：

> 它仍然是主力素材，但已经进入衰退预警，需要保留现金流，同时停止继续加压，并立刻补同角度变体。

### 4. 这是不是素材疲劳，而不是受众或页面问题

素材衰退管理里最常见的误判，是把所有表现下滑都归因到素材。

我会先拆三种情况：

| 问题 | 典型表现 | 优先动作 |
|---|---|---|
| 素材疲劳 | Frequency 上升，CTR 下降，ROAS / CPA 变差 | 换 hook、封面、前三秒、剪辑节奏 |
| 受众饱和 | 新旧素材都难跑，Reach 增速变慢 | 扩市场、扩受众、换产品包或调整结构 |
| 漏斗承接问题 | CTR 正常，但 CVR / ROAS 下滑 | 查落地页、价格、库存、优惠、支付、评论 |

如果 CTR 没掉，但 CVR 掉了，我不会优先说“素材疲劳”。这更可能是页面、价格、库存、履约、优惠或归因窗口的问题。

### 5. 新素材是不是真的有差异

素材供给不是数量问题，而是有效差异问题。

如果一周上了 20 条素材，但都是同一个脚本换字幕、换封面、换剪辑节奏，我不会把它当成 20 个有效测试。

我会看这些差异：

| 维度 | 我想验证什么 |
|---|---|
| 购买理由 | 痛点、结果、优惠、场景、身份、风险解除 |
| Hook | 用户为什么愿意停下来 |
| 画面 | UGC、测评、开箱、前后对比、场景演示 |
| 达人 | 身份、表达方式、可信度、受众匹配 |
| 证明方式 | 评论、结果、过程、对比、数据、第三方背书 |

真正值得沉淀的不是“这条视频跑得好”，而是：

> 哪个购买理由、哪种表达结构、哪类达人和哪种证明方式更容易带来稳定转化。

### 6. 素材库有没有形成接力机制

素材管理最后一定要回到供给节奏。

我会把素材分成 5 个状态：

| 状态 | 判断 | 动作 |
|---|---|---|
| 测试中 | 上线时间短或样本不足 | 继续观察 |
| 潜力素材 | 前链路好，但 GMV 未放大 | 小预算复测或换结构 |
| 放量素材 | ROAS / CPA 达标，GMV 占比上升 | 保留，进入主力池 |
| 衰退预警 | 频次、CTR、ROAS、CPA 出现组合恶化 | 补变体，不继续加压 |
| 退役素材 | 连续低效且无局部信号 | 暂停，沉淀标签和原因 |

素材库健康，不是看“有多少素材文件”，而是看：

- 有多少素材已经通过测试；
- 有多少素材可以替换头部素材；
- 有多少素材能覆盖不同购买理由；
- 有多少素材可以在未来 7-14 天内进入测试；
- 退役素材有没有沉淀出可复用经验。

## 我的日常判断规则

我不会把下面这些阈值当成自动关停规则。它们更适合作为看板预警，最后仍然要结合样本量、归因窗口、预算和业务目标判断。

| 等级 | 触发条件 | 我会怎么处理 |
|---|---|---|
| 黄色 | Top3 GMV 占比 > 60% 或 Top1 GMV 占比 > 30% | 准备同角度变体，先补素材，不直接替换赢家 |
| 橙色 | Frequency 上升，ROAS 衰减率 <= -20% | 刷新 hook、封面、前三秒、达人剪辑版本 |
| 红色 | CTR 下滑 >= 20%，CPA 上升 >= 20%，ROAS 低于目标 | 降预算或暂停，复查样本量、归因和落地页 |
| 黑色 | Top3 同时下滑，且没有测试通过的新素材 | 停止放量，集中补素材库存 |

样本量不足时，我会降低结论强度。

至少要先确认：

- 近 7 日花费是否达到目标 CPA 的 2-3 倍；
- 近 7 日是否有足够 Purchase 或有效中间行为；
- 归因窗口是否一致；
- 是否刚经历大促、缺货、页面改版或支付异常；
- 同一素材是否被多个 ad id 拆散统计。

## 我会如何把它用在 Meta / TikTok / Google

### Meta Ads

Meta 的素材管理更像是“用素材打开人群”。素材本身会影响系统理解谁可能购买。

我会重点看：

- Broad / ASC 里主力素材是不是过度集中；
- 头部素材掉量前有没有同角度变体；
- 新素材是否真的测试了不同购买动机；
- CTR 下滑时是 hook 问题，还是受众重复曝光问题；
- 再营销 ROAS 是否掩盖了新客素材不足。

### TikTok Ads

TikTok 的素材衰退通常更快，更依赖持续上新。

我会重点看：

- 前 3 秒是否开始失效；
- 视频节奏和字幕密度是否还适合当前流量；
- daily new users 是否下降；
- 达人表达是否同质化；
- 是否只是复制爆款形式，而没有复制背后的购买理由。

### Google Demand Gen / PMax

Google 的素材替换要更谨慎。

我不会看到低评分 asset 就直接删除。更稳的方式是先补新素材，再逐步替换，并且区分 Demand Gen 的视觉种草任务和 PMax 的商品 / 资产组合任务。

我会重点看：

- PMax asset group 是否按产品线、场景或人群组织；
- Demand Gen 是否有足够视频、图片和短文案资产；
- 低表现 asset 是否真的拖累整体，还是仍有边际贡献；
- 替换素材是否会干扰常青 campaign 的学习稳定性。

## 常见误判

### 1. 把老素材等同于疲劳素材

老素材不一定疲劳。只要 ROAS、CPA、CTR、CVR 仍然稳定，它仍然是现金流素材。正确动作是补变体，而不是粗暴暂停。

### 2. 只看 GMV 加权日龄

GMV 加权日龄能看主力成交是否偏老，但它会低估花费高、GMV 已经下降的素材。所以必须同时看花费加权日龄和曝光加权日龄。

### 3. Top3 占比低就一定健康

Top3 低可能代表抗风险强，也可能代表没有真正赢家。要结合 ROAS、CPA、有效素材数和素材标签看。

### 4. CTR 掉了就一定换素材

CTR 下滑可能是素材疲劳，但也可能是受众重复曝光、流量结构变化、竞价环境变化。需要结合 Frequency、Reach 和预算变化判断。

### 5. CVR 掉了也归因到素材

如果 CTR 没问题但 CVR 掉了，更应该先查落地页、价格、库存、优惠、物流、评论、支付和归因窗口。

### 6. 把同质化变体当素材供给

同一个脚本换字幕，不等于新创意。素材管理要沉淀不同购买理由，而不是只堆数量。
