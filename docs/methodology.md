# 素材衰退管理方法论

## 1. 先定义“素材衰退”

素材衰退不是“素材上线时间变长”，而是同一素材或高度相似素材在目标人群里被反复看到后，点击和转化响应下降，最终表现为 CTR/CVR/ROAS 下滑或 CPA 上升。

专业管理上要区分三类问题：

| 问题 | 典型现象 | 处理方式 |
|---|---|---|
| 素材疲劳 | 同素材频繁曝光，CTR/CVR 下降 | 加入新创意、换 hook、换剪辑、换封面 |
| 受众饱和 | 新旧素材都跑不动，Reach 增速下降 | 扩受众、换市场、换产品包、调整出价/预算 |
| 漏斗问题 | CTR 正常但 CVR/ROAS 下滑 | 查落地页、价格、库存、优惠、支付、归因 |

## 2. 年龄指标

只看平均日龄会误判，因为一条无花费素材的年龄不应该影响主判断。因此建议同时看三个加权日龄：

| 指标 | 公式 | 用途 |
|---|---|---|
| GMV 加权日龄 | `SUM(日龄 * GMV) / SUM(GMV)` | 当前 GMV 是否由老素材贡献 |
| 花费加权日龄 | `SUM(日龄 * Spend) / SUM(Spend)` | 预算是否压在老素材上 |
| 曝光加权日龄 | `SUM(日龄 * Impressions) / SUM(Impressions)` | 用户是否主要看到老素材 |

判断逻辑：

| 情况 | 解释 | 动作 |
|---|---|---|
| GMV 加权日龄高，ROAS 稳定 | 老赢家仍健康 | 保留，不要粗暴替换，同时准备变体 |
| 花费加权日龄高，GMV 加权日龄低 | 老素材吃预算但贡献下降 | 查预算浪费，考虑降预算或替换 |
| 曝光加权日龄高，CTR 下滑 | 重复曝光压力上升 | 刷新 hook、封面、前三秒 |

## 3. 集中度指标

原始 R3 建议命名为 `Top3 GMV 占比` 或 `Top3 Concentration`，含义更清晰：

```text
Top3 GMV 占比 = Top3 素材 GMV 之和 / 总 GMV
```

补充两个指标：

```text
HHI = SUM(每条素材 GMV 占比 ^ 2)
有效素材数 = 1 / HHI
```

| 组合状态 | 判断 | 风险 |
|---|---|---|
| Top3 > 70% | 极度依赖头部素材 | 爆款掉量会造成 GMV 大幅波动 |
| Top3 45%-65% | 通常较健康 | 有赢家，也有替补 |
| Top3 < 40% | 贡献分散 | 抗风险强，但可能缺少强爆款 |
| 有效素材数 < 3 | 实际靠 1-3 条素材 | 需要优先补同角度变体 |

## 4. 衰减指标

建议用近 3 日对比近 7 日，而不是只看昨天。单日数据受预算、竞价、归因延迟影响大。

| 指标 | 公式 | 判断 |
|---|---|---|
| ROAS 衰减率 | `近3日ROAS / 近7日ROAS - 1` | <= -20% 进入预警 |
| CTR 衰减率 | `近3日CTR / 近7日CTR - 1` | <= -20% 说明吸引力走弱 |
| CPA 上升率 | `近3日CPA / 近7日CPA - 1` | >= 20% 说明成本压力上升 |

注意：如果花费很低、购买数很少，不要直接判定衰退。建议至少满足以下任一条件再做动作：

- 近 7 日花费达到目标 CPA 的 2-3 倍；
- 近 7 日购买数达到 5-10 单；
- 近 7 日曝光达到该账户正常判断所需的最小量级。

## 5. 素材状态分层

| 状态 | 判断 | 操作 |
|---|---|---|
| 测试中 | 上线 < 3 天或样本不足 | 继续观察 |
| 潜力素材 | CTR/CVR 好，但 GMV 未放大 | 小预算继续测，换结构或受众 |
| 放量素材 | ROAS/CPA 达标，GMV 占比上升 | 保留，补同角度变体 |
| 衰退预警 | 频次上升且 ROAS/CTR 下滑 | 刷新素材元素 |
| 退役素材 | 连续 3-5 天低于目标且无恢复 | 暂停或降预算，沉淀标签 |

## 6. 每日 SOP

1. 先看 `portfolio_metrics.csv`：
   - Top1 GMV 占比是否超过 30%；
   - Top3 GMV 占比是否超过 60%；
   - GMV/花费/曝光加权日龄是否连续上升；
   - 有效素材数是否低于 3-4。

2. 再看 `creative_scores.csv`：
   - 排查 spend_share 高但 ROAS 低的素材；
   - 排查 frequency 高且 ROAS 衰减的素材；
   - 保留 r3_roas 达标且 gmv_share 高的主力素材；
   - 对主力素材建立同角度变体任务。

3. 输出动作：
   - 立即执行：降预算、暂停、补同角度变体；
   - 观察等待：样本不足但趋势不差；
   - 小预算测试：新 hook、新达人、新场景；
   - 需要确认：归因窗口、落地页、优惠、库存、素材授权。

## 7. 周度复盘

每周至少按素材标签复盘：

| 维度 | 复盘字段 |
|---|---|
| 角度 | pain point、testimonial、comparison、offer、demo |
| 画面 | UGC、测评、开箱、场景、前后对比、产品特写 |
| Hook | 前 3 秒话术、冲突点、利益点、问题点 |
| 达人 | creator_name、授权状态、Post ID、投流码 |
| 平台 | Meta、TikTok、Google Demand Gen、PMax |

最终目标不是“避免所有素材衰退”，而是让账户始终有足够的新素材接力，降低 GMV 对少数爆款的依赖。
