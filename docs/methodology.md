# 素材衰退管理方法

这份文档是 README 的展开版，用来说明我会如何把素材衰退从“单条素材表现问题”拆成“素材组合运营问题”。

我不把素材衰退简单理解成素材上线时间变长。更准确地说，它是素材、受众、预算、转化链路和素材补给共同作用后的结果。

如果只看一条素材今天 ROAS 掉了没有，很容易做出错误动作。比如：

- 老素材还有利润，却因为日龄高被误杀；
- 新素材样本不足，却被太早判断为失败；
- 页面和 offer 出问题，却被误判成素材疲劳；
- 账户已经高度依赖 1-3 条素材，但团队还以为整体 ROAS 稳定。

所以这套方法的重点不是“找到一个自动关停公式”，而是建立一套素材判断语言。

---

## 1. 先判断问题属于哪一类

素材表现下滑不一定都是素材疲劳。

我会先把问题分成三类：

| 问题 | 典型现象 | 我会先查什么 |
|---|---|---|
| 素材疲劳 | 同一批素材重复曝光，CTR / CVR / ROAS 下滑 | Frequency、Reach、CTR、创意相似度 |
| 受众饱和 | 新旧素材都难跑，Reach 增速变慢，CPM 或 CPA 上升 | 受众规模、预算、市场、国家、排除规则 |
| 漏斗承接问题 | CTR 正常，但 CVR、CPA、ROAS 变差 | 落地页、价格、库存、优惠、支付、评论、归因 |

这个拆分很重要。

如果是素材疲劳，应该刷新 hook、封面、剪辑、达人和证明方式。  
如果是受众饱和，单纯换素材可能解决不了，需要看市场、受众、预算和产品组合。  
如果是漏斗承接问题，再多素材也可能只是把更多流量送到一个接不住的页面。

## 2. 用加权日龄看素材组合是否变老

如果你第一次看加权日龄公式，可以先读 [`docs/formula-explainer.md`](./formula-explainer.md)。里面用具体数字解释了 GMV 加权日龄、花费加权日龄、曝光加权日龄、Top3 GMV 占比和 HHI。

平均日龄本身意义不大。

因为一条没有花费、没有 GMV 的素材，即使已经上线 100 天，也不应该影响主判断。

我会看三个加权日龄：

| 指标 | 公式 | 它回答的问题 |
|---|---|---|
| GMV 加权日龄 | `SUM(日龄 * GMV) / SUM(GMV)` | 当前成交是不是由老素材贡献 |
| 花费加权日龄 | `SUM(日龄 * Spend) / SUM(Spend)` | 当前预算是不是压在老素材上 |
| 曝光加权日龄 | `SUM(日龄 * Impressions) / SUM(Impressions)` | 用户是不是主要看到老素材 |

这三个指标要一起看：

| 情况 | 判断 | 动作 |
|---|---|---|
| GMV 加权日龄高，ROAS 稳定 | 老赢家仍然健康 | 保留，不要粗暴替换 |
| 花费加权日龄高，GMV 加权日龄低 | 老素材吃预算但贡献下降 | 排查预算浪费 |
| 曝光加权日龄高，CTR 下滑 | 重复曝光压力上升 | 刷新素材开头和视觉 |
| 三个日龄都上升，新素材贡献低 | 素材补给断层 | 优先补素材，而不是硬放量 |

日龄不是关停依据，而是结构预警。

## 3. 用集中度看账户有没有单点风险

原始的 R3 指标，我更建议命名为 `Top3 GMV 占比`。

```text
Top3 GMV 占比 = Top3 素材 GMV 之和 / 总 GMV
```

这个名字更直接，也能避免和近 3 日指标、留存指标或其他 R3 口径混淆。

我会同时看：

| 指标 | 公式 | 用途 |
|---|---|---|
| Top1 GMV 占比 | `Top1 GMV / 总 GMV` | 判断是否依赖单条素材 |
| Top3 GMV 占比 | `Top3 GMV 之和 / 总 GMV` | 判断头部集中风险 |
| HHI | `SUM(每条素材 GMV 占比 ^ 2)` | 判断整体集中度 |
| 有效素材数 | `1 / HHI` | 估算真正支撑 GMV 的素材数量 |

我会这样理解：

| 状态 | 判断 |
|---|---|
| Top3 > 70% | 高风险，账户高度依赖头部素材 |
| Top3 45%-65% | 通常比较健康，有赢家，也有替补 |
| Top3 < 40% | 贡献分散，但也可能缺少强爆款 |
| 有效素材数 < 3 | 表面素材多，实际贡献集中 |

集中不是坏事。

投放里一定会有赢家。问题不是“有没有赢家”，而是“赢家掉量时有没有接力素材”。

## 4. 用近 3 日和近 7 日看衰退趋势

单日表现很容易受预算、竞价、归因延迟、大促、库存和页面变化影响。

所以我更倾向于用近 3 日对比近 7 日：

| 指标 | 公式 | 解释 |
|---|---|---|
| ROAS 衰减率 | `近3日ROAS / 近7日ROAS - 1` | 收入效率是否下滑 |
| CTR 衰减率 | `近3日CTR / 近7日CTR - 1` | 素材吸引力是否下降 |
| CPA 上升率 | `近3日CPA / 近7日CPA - 1` | 获客成本是否恶化 |

我会把衰退判断分成几层：

| 表现 | 判断 |
|---|---|
| ROAS 小幅下滑，但样本不足 | 继续观察 |
| CTR 下滑，Frequency 上升 | 高概率素材疲劳 |
| CTR 稳定，CVR 下滑 | 优先查页面、价格和 offer |
| 主力素材 ROAS 衰减 > 20%，CPA 上升 > 20% | 主力衰退预警 |
| Top3 同时下滑 | 账户级风险，不只是单素材问题 |

判断前要先看样本量。

我通常会要求至少满足以下任一条件，再做强动作：

- 近 7 日花费达到目标 CPA 的 2-3 倍；
- 近 7 日购买数达到 5-10 单；
- 近 7 日曝光达到该账户日常判断所需的最小量级；
- 该素材已经完成平台学习期或有稳定投放历史。

## 5. 把素材分成可运营状态

素材管理不能只有“开”和“关”。

我会把素材分成 5 种状态：

| 状态 | 判断 | 动作 |
|---|---|---|
| 测试中 | 上线短、样本不足 | 继续观察，避免过早判断 |
| 潜力素材 | CTR / CVR 有信号，但 GMV 未放大 | 小预算复测或换结构 |
| 放量素材 | ROAS / CPA 达标，GMV 占比上升 | 保留，进入主力池 |
| 衰退预警 | 频次、CTR、ROAS、CPA 出现组合恶化 | 补变体，减少继续加压 |
| 退役素材 | 连续低效且无局部信号 | 暂停，记录失败原因 |

这样做的好处是，素材复盘不会变成简单的“好 / 不好”。

一条素材可能不适合继续放量，但适合二创。  
一条素材可能当前没跑出来，但证明了某个购买理由值得继续测。  
一条素材可能已经退役，但它的 hook、场景或达人表达仍然值得沉淀。

## 6. 每日看板应该怎么读

我会先看组合，再看单条素材。

### 第一步：看组合风险

先看：

- Top1 GMV 占比；
- Top3 GMV 占比；
- HHI；
- 有效素材数；
- GMV / 花费 / 曝光加权日龄；
- 新素材 GMV 占比；
- 近 7 天新增素材数和通过测试素材数。

这一步的目标，是判断账户是否已经被少数素材控制。

### 第二步：看主力素材是否进入衰退

再看：

- 主力素材近 3 日 ROAS；
- 主力素材近 7 日 ROAS；
- CTR 衰减率；
- CPA 上升率；
- Frequency；
- GMV 占比和花费占比。

这一步的目标，是判断头部素材是不是还健康。

### 第三步：看新素材是否能接力

最后看：

- 新素材有没有拿到足够曝光；
- 新素材是不是只在换皮，没有测试新购买理由；
- 哪些新素材前链路好但后链路弱；
- 哪些素材值得进入主力放量池；
- 哪些素材需要二创或补证言。

这一步的目标，是判断未来 7-14 天账户有没有续航。

## 7. 周度复盘应该沉淀什么

周度复盘不是只汇总 Top 素材。

我更关心这些问题：

| 维度 | 复盘问题 |
|---|---|
| 购买理由 | 哪些痛点、场景、offer、证明方式更容易带来转化 |
| Hook | 哪类开头更容易带来 CTR 和观看停留 |
| 达人 | 哪类身份、表达、语气和人设更容易建立信任 |
| 画面 | UGC、测评、对比、开箱、场景演示哪个更有效 |
| 平台 | Meta、TikTok、Google Demand Gen / PMax 的素材反馈有什么差异 |
| 生命周期 | 哪类素材衰退更快，哪类素材能撑更久 |
| 复用价值 | 哪些素材适合二创、复拍、换平台或进入常青素材库 |

好的素材复盘不应该只留下“这条素材效果好”。

更有价值的结论应该是：

> UGC 痛点前置 + 真实结果证言，在美国 Broad 里能稳定拿到更低 CPA，但生命周期较短。下一轮应该保留痛点结构，换达人和前三秒视觉，同时补一组更强的落地页证据承接。

这样的结论才会变成可复用经验。

---

## 8. 这套方法的边界

这套方法不能替代投放判断。

它更像一个素材管理框架，帮助团队减少三类错误：

1. 误杀还在赚钱的老素材；
2. 放任已经衰退的主力素材继续吃预算；
3. 等到头部素材掉量后才发现没有新素材接力。

需要人工确认的地方包括：

- `creative_id` 是否真的代表同一条素材；
- 同一素材是否被多个 ad id 或 campaign 拆散；
- GMV 是否采用同一归因窗口；
- Google PMax 是否能拿到足够细的 asset / asset group 数据；
- 素材日龄是首次上线日期，还是广告创建日期；
- 大促、缺货、页面改版、支付异常是否影响了数据；
- Ref 码、Post ID、达人授权和 UTM 是否完整。

如果这些口径不清楚，指标再漂亮也可能误导判断。
