# 优化师审校记录

## 结论

原始思路“加权日龄 + Top3 GMV 集中度”方向正确，但不能单独作为素材衰退判断。它更适合做素材组合风险预警，而不是直接决定素材是否暂停。

本项目做了以下修正：

| 原内容 | 风险 | 修正 |
|---|---|---|
| 只看 GMV 加权日龄 | 会低估“花费高但 GMV 掉”的素材 | 增加花费加权日龄和曝光加权日龄 |
| R3 命名不够清楚 | 容易和其他 R3/留存/近 3 日指标混淆 | 改为 Top3 GMV 占比，保留原公式 |
| 高 CR3 被描述为“易于管理” | 管理上确实简单，但商业风险更高 | 明确为高集中、高波动风险 |
| 低 CR3 被描述为“抗风险强” | 可能只是没有赢家，也可能是预算太散 | 增加 HHI 和有效素材数辅助判断 |
| 缺少疲劳原因拆分 | 素材疲劳、受众饱和、漏斗问题会混淆 | 加入问题分层和检查路径 |
| 缺少样本量要求 | 小样本容易误杀新素材 | 加入花费、购买数、曝光的判断前提 |

## 专业判断

更稳的做法不是固定素材生命周期，而是素材组合管理：

1. 用加权日龄判断主力素材是否偏老。
2. 用 Top1 / Top3 / HHI 判断账户是否过度依赖少数素材。
3. 用近 3 日 vs 近 7 日 ROAS、CTR、CPA 判断是否真的衰退。
4. 用 Frequency、Reach、daily new users 判断是否有重复曝光压力。
5. 用每周新增素材数和通过测试素材数判断未来是否有接力。

## 推荐阈值

| 指标 | 健康区间 | 预警区间 | 高风险区间 |
|---|---|---|---|
| Top1 GMV 占比 | < 25% | 25%-35% | > 35% |
| Top3 GMV 占比 | 45%-65% | 65%-70% | > 70% |
| 有效素材数 | > 5 | 3-5 | < 3 |
| ROAS 衰减率 | > -10% | -10% 到 -20% | <= -20% |
| CTR 衰减率 | > -10% | -10% 到 -20% | <= -20% |
| CPA 上升率 | < 10% | 10%-20% | >= 20% |

需确认：不同品类、国家、客单价和预算等级下，阈值需要回测校准。以上阈值适合作为默认看板预警，不适合作为自动关停规则。

## 对投放动作的补充

| 情况 | 动作 |
|---|---|
| 老素材 ROAS 稳定且 GMV 占比高 | 保留，补同角度变体，不要直接替换 |
| 花费占比高但 GMV 占比低 | 优先检查预算浪费 |
| 频次高、CTR 下滑、ROAS 下滑 | 高概率素材疲劳，刷新视觉和开头 |
| CTR 稳定但 CVR/ROAS 下滑 | 不优先怪素材，先查站点和优惠 |
| Top3 同时下滑 | 账户级风险，暂停放量并集中补素材 |

## 上传前人工确认

如果把这套模型用于 Meta / Google / TikTok 批量管理，建议人工确认：

- `creative_id` 是否能唯一代表素材；如果同素材有多个 ad id，需要做归并；
- `launch_date` 是素材首次上线日期，不是广告组或 campaign 日期；
- GMV 是否采用同一归因窗口；
- Google PMax 是否能拿到可用的 asset / asset group 维度数据；
- Post ID、达人授权、投流码、Ref 码和 UTM 是否完整；
- 样本量是否达到判断阈值，避免误杀学习期素材。
